# Helionis - Mystical Spiritual E-Commerce Website 🌟

Eine vollständig funktionsfähige, mystische E-Commerce-Website für spirituelle Produkte mit modernem Glassmorphism-Design und umfassender Funktionalität.

## 🎨 Design Features

- **Mystisches Design**: Dunkler Rauch-Hintergrund mit animierten Partikeln
- **Glassmorphism**: Moderne Glas-Effekte mit Backdrop-Blur
- **Responsive Design**: Optimiert für alle Geräte
- **Animationen**: Schwebende Rauch-Effekte und mystische Partikel
- **Farben**: Mystische Grüntöne, Silber und Gold-Akzente

## 🚀 Installation & Setup

### 1. Repository klonen
```bash
git clone <repository-url>
cd helionis-website
```

### 2. Dependencies installieren
```bash
npm install
```

### 3. Firebase konfigurieren
1. Erstelle ein neues Firebase-Projekt
2. Aktiviere Authentication, Firestore und Hosting
3. Ersetze die Konfiguration in `firebase-config.js`

### 4. Lokale Entwicklung starten
```bash
npm start
# oder
firebase serve
```

### 5. Deployment
```bash
firebase deploy
```
